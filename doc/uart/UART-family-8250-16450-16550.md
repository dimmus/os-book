# 8250 vs 16450 vs 16550 (quick family notes)

This is a **short companion** to **`PC16550D.pdf`** in this directory—not a replacement for the full datasheet.

| Part | FIFO | Typical notes |
| --- | --- | --- |
| **8250** | No | Original PC UART; several errata on early chips (8250 vs 8250A). |
| **16450** | No | Faster than original 8250 in same programming model; still **no** FIFO. |
| **16550** | Yes (16-byte TX/RX typically) | **FIFO** + compatible legacy registers; what **QEMU** and most PC Super I/O emulate for COM ports. |

**Programming:** same **I/O port offsets** from `COM1_BASE` (`0x3F8`) for **LCR**, **LSR**, **baud divisor** (with **DLAB** in **LCR**), etc. The **`serial.hpp`** driver is written for a **16550-class** UART (**FCR** used to enable/clear FIFOs).

**Baud:** divisor **N** in **DLL/DLM** with **DLAB=1**; **baud ≈ f_reference / (16 × N)** (see datasheet for exact clocking).

For exhaustive bit definitions and electrical limits, use **`PC16550D.pdf`**.
